Lista de configuraciones aplicadas en la aplicación listas para probar:

  • Para ejecutar el controlador que lanza la vista: saludar.html pon en la URL: saludar/nombre_que_quieras
  • Para ejecutar el controlador que lanza la vista: saludarParam.html pon en la URL: saludoParam?nombre=nombre_que_quieras o sin nada. Será valor por defecto
  • Para ejecutar un formulario que lanza la vista: formulario.html pon en la URL: /producto/form
